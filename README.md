# Simple todo app
This project represents a simple todo app written in React.

See the **[demo](https://matobeno1-sample-todo-app.netlify.app/)**.

## Usage
Users can:
- create todo
- update todo
- delete todo
- (un)mark todo as complete
- (un)mark all todos as complete
- delete completed todos only

## Stack
- react (CRA)
- redux
  - redux toolkit
- reselect
- antd
